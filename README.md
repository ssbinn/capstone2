## Colab_Subway
익명화 기술(LDP)을 이용하여 안전한 교통 빅데이터 분석 어플리케이션<br>

개인 정보 비식별화 기술인 LDP를 이용하여 개인의 프라이버시를 보존할 수 있습니다.

<br>

## 프로젝트 소개
메인 메뉴 화면과 원본 그래프 비교 화면 <br>

### 주요 기능
- 원본 승하차 정보와 노이즈가 섞인(LDP가 적용된) 승하차 정보를 나타낸 비교 그래프 
- 역별 지하철 혼잡도 확인
- 요일별, 시간별 교통 체증 확인
